# 🚀 Guide Rapide - Mode Offline GestiStock

## Pour l'utilisateur final

### Comment ça marche?

#### 1. Connexion normale (en ligne)
- Vous utilisez l'app normalement
- Toutes les données sont automatiquement sauvegardées localement
- Rien ne change pour vous

#### 2. Perte de connexion
- Un banner orange apparaît en haut: **"Mode hors ligne"**
- Vous pouvez toujours:
  - ✅ Voir tous vos produits
  - ✅ Consulter les clients
  - ✅ Voir les ventes
  - ✅ Créer de nouveaux produits
  - ✅ Enregistrer des ventes
  - ✅ Modifier des données

#### 3. Modifications hors ligne
- Toutes vos actions sont enregistrées
- Le banner affiche: **"3 action(s) en attente"**
- Rien n'est perdu!

#### 4. Retour de connexion
- Le banner devient bleu: **"Synchronisation en cours..."**
- Toutes vos modifications sont envoyées automatiquement
- Message de succès: **"3 action(s) synchronisée(s)"**
- Le banner disparaît
